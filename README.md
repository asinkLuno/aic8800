# AIC8800 Driver for Linux 6.18+

本仓库基于绿联(Ugreen)的 AIC8800 驱动，适配 Linux 内核 6.18 及以上版本。

## 背景

AIC8800 是一款 WiFi 6 无线网卡芯片，绿联官方提供的驱动主要针对旧版本 Linux 内核。本项目在官方驱动基础上进行了修改，以支持 Linux 6.18+ 内核。

## 修改内容

- 修复内核 API 变更导致的编译错误
- 适配新的内核网络子系统接口
- 兼容内核 6.18+ 的模块加载机制

## 使用说明

详细安装步骤请参考：[AX300 驱动安装教程_Linux_V4.0.pdf](AX300%20驱动安装教程_Linux_V4.0.pdf)

## 兼容性

- Linux 内核版本：6.18+
- 芯片型号：AIC8800 (AX300)

## 许可证

本驱动基于绿联官方源码修改，遵循原许可证。
